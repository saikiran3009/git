# git

--- first commit and push

git config user.name "====="
git config user.email "===="

git add . (to add the file) & git add file_name(if new file created - because github could not track the local file)
git status (to check the status of the file(red-file not added)-(green-file added))
git commit -m "write the message"
git push (to push the file into github)

git log --oneline (to view logs)


--- Branching 

git branch -a (to check the existing branching)
git fetch (to fetch the newly created branches in GitHub)
git checkout branch_name (To enter the new branch)
